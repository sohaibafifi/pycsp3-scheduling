## Quick Start Notebook

The **Getting Started** tutorial covers:

### Part 1: API Reference
- IntervalVar creation and properties
- IntervalVarArray and IntervalVarDict factories
- SequenceVar for resource sequencing
- Precedence constraints (8 types)
- Grouping constraints (span, alternative, synchronize)
- Cumulative functions (pulse, step_at_*)
- State functions and transition matrices

### Part 2: Interactive Examples
1. **Job Shop Scheduling** - Classic machine scheduling
2. **RCPSP** - Resource-constrained project scheduling
3. **Flexible Job Shop** - Alternative machine selection
4. **Setup Times** - Sequence-dependent transitions
5. **Multi-Resource** - Tasks using multiple resources
6. **Span & Synchronize** - Task hierarchies and meetings

```{tip}
Download the notebook and run it locally to experiment with the examples!
```
